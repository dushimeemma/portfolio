## Tech Stack

**Client:** NextJs

## Installation

Install portfolio using yarn

```bash
  git clone
  cd portfolio
  yarn install
  yarn dev
```

## Running Tests

To run tests, run the following command

```bash
  yarn test
  yarn test:coverage
```
